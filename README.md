# React під контролем: Controlled та Uncontrolled компоненти

Навчальний проєкт на React, створений за допомогою Vite.  
Демонструє роботу з контрольованими та неконтрольованими формами, а також асинхронний запит до сервера з використанням `useEffect`.

## Компоненти

- `ControlledForm` — контрольована форма (усі значення керуються через state).
- `UncontrolledForm` — неконтрольована форма (значення читаються через `useRef`).
- `PostsFetcher` — компонент, який завантажує дані з API та обробляє стани завантаження, успіху та помилки.
